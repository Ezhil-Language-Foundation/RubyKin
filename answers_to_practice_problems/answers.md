குறிப்பு : collection இற்குள் collections களை நாம் வைத்திருக்கலாம் என்பதை நினைவில்  வைத்திருங்கள்.

  ```ruby
  => 9
  bag["potions"]["blue"] = bag["potions"]["blue"] - 2
  => 3
  bag
  => {
    "frogs" => 1,
    "herbs" => 2,
    "scrolls" => 9,
    "potions" => {
      "orange" => 4,
      "blue" => 3,
      "red" => 10
      }
    }
  ```
